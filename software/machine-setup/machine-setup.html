This guide has been developed to walk you through the required installations to contribute to our codebase.

Our robot software is primarily written in C++, with addition to the ROS Library. This system is much easier to compile in Linux, but development can be done in both Linux and Windows. Start with the Developer Machine Setup section and continue through until you reach the end of the section.
